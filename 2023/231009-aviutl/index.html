AviUtl 入门踩坑全记录

2023/10/09 字数统计： 2k 阅读时间： 3 mins 访问次数： ... 评论统计： ...
  • 学习笔记
  • AviUtl

　　喜报：截至发出时间我已经用 Aviutl 做了三个视频了，效果喜人，虽然安装的过程不可谓不折腾……
　　总之以下是一些踩坑记录：

安装

　　本来在 github 下好装好开开心心结果用的时候发现很多插件迷之消失，只能自己去下。然而日文原版的插件用中文系统打开会乱码，用 ntleas 也转码不了已经加密过的文件，更是发现了重度恶性 bug：载入 patch.aul 时扩展编辑窗口中的播放线不会移动，无论怎么设置都没有用。
　　聪明如我（……）自然第一时间就想到应该是汉化导致的目标名不匹配，然而将 patch.aul 自带的配置文件全部设为 false 也解决不了，看着 B 站上其他人都可以正常使用我突然冒出一个念头：该不会是我下载的不对吧？
　　然后事实证明，真的是我下载的不对……！Github 版本的安装包有非常多的插件来源是错误 / 丢失的，而且每次装出来的东西根据网络情况会不一样，非常神奇，可能还会混杂一些 拓展编辑 r92 无法支持的插件。总之在百度云下载汉化插件后一切的问题都解决了……
　　我还能说什么，我只能感慨一下国内分享软件的现状 :4o-2:

界面美化

夜间模式

依赖：patch.aul + DarkenWindow
Ps. Github 直接下载版为日文编码，汉化软件使用者请前往 B 站下载汉化插件

　　解压后将 patch.aul 放置在 AviUtl 根目录下，DarkenWindow 则是将 readme 中写到的 cc6.manifest、DarkenWindow.aul 与 DarkenWindow 文件夹放入 AviUtl 目录内的 plugins 文件夹中。
　　安装完毕后在主窗口菜单栏右键即可看到主题选项，除了夜间模式以外还有一些非常千禧风格的配色，理论上还可以自己写。

　　还有一个插件可以把 AviUtl 窗口合并做成 Adobe 全家桶那样：【AviUtl】ワークスペースを作成できるプラグイン【SplitWindow】
　　但是我个人觉得 AviUtl 这个拆分窗口的设计做得还挺别致的，只放出来做个记录吧。

编辑优化

播放 / 暂停

　　AviUtl 默认设置是在播放中按下空格后会跳回开始播放的位置，下次再播放从同一帧开始，这个设计在浏览动画特效的时候很方便，但在做字幕的时候就比较难受了。
　　修改的方式是右键 拓展编辑 窗口（本质就是图层时间轴啦）进入 环境设置，勾选 播放窗口播放时与光标同步 与 光标移动时自动滚动 即可。
　　Ps. 如果勾选了还无效大概需要返回上文安装 patch.aul

x264 导出

依赖：x264guiEx
这个似乎不用汉化，直接从 github 下载就能用

　　下载后将所有文件拖入根目录，导出视频时使用 文件→插件渲染→x264 扩展渲染 即可。

显示音频波形

依赖：ShowWaveform
未测试原版是否兼容但建议下载汉化版

　　下载后依照 readme 将 ShowWaveform.auf 与 ShowWaveform.ini 放入 AviUtl 目录内的 plugins 文件夹中。
　　似乎不会自动生效，需要每次启动后手动进入 设置→item 内音频波形设置 菜单点击刷新按钮后才会出现。

滤镜拖动排序

依赖：DragFilter
未测试原版是否兼容但建议下载汉化版

　　适用于所有自带滤镜与动画效果的拖动排序。
　　自己看 readme，说累了（累了）。
　　吐槽一下这个插件的复制滤镜功能居然不能跨 object 实现而是在同一个 object 内复制滤镜……不是，同一个 object 里挂一串一模一样的滤镜是要干嘛？

跨物件复制滤镜

依赖：CopyFilter
未测试原版是否兼容但建议下载汉化版

　　适用于所有自带滤镜与动画效果的复制与粘贴，可以在不同物件中操作，非常适合做字幕时使用。
　　安装方式看 readme。
　　选中最上方的滤镜右键选择 复制下方滤镜 即可复制该物体的全部滤镜，在新物体的详情窗口下方右键粘贴即可。

操作小技巧

多选

　　按住 Ctrl 键可点选 / 框选时间轴（即 扩展编辑）内物体。
　　为什么加粗呢是因为一旦松开 Ctrl 键多选就失效了【。
　　在按住 Ctrl 的同时可以用鼠标批量移动物体，也可以批量修改详情窗口里的属性，可以用于做字幕时批量修改字体和颜色。

锁定 / 新建图层

　　右键时间轴（即 扩展编辑）左侧的图层名勾选 锁定图层。
　　选择 新建图层 会将所有图层下移一行，需要注意的是 AviUtl 的操作逻辑是图层靠下的覆盖图层在上的，因此这个功能其实并不常用。

新建合成

　　类似 AE 的功能。在 扩展编辑 左上角点击 Root 会出现一个场景下拉菜单，选中空白场景即可进入新合成。
　　右键点击可设置场景的画幅尺寸与场景名。

导入合成

　　在时间轴空白处右键选择 添加媒体物件→场景 即可导入其它合成，默认是静音状态，如需同步插入音频需要选择 场景（音频）选项。
　　Root 场景无法插入其它场景。
　　可以单独导出某一个场景的视频，在某一段画面用了特别复杂的滤镜时可以考虑通过「提前导出，以视频的形式加入 root 场景」的方式减轻系统负担。

时间轴辅助线

　　在时间轴空白处右键勾选 网格显示（BPM），进入 网格设置 后填入 BPM 与节拍即可在时间轴上创建节奏网格，非常方便 BGM 踩点。
