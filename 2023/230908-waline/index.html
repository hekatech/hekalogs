服务器端禁用盒武器

　　然后 waline v2.x 就直接上线了显示用户 IP 的功能。
　　你™是不是有病啊？窥私癖？
　　不要跟我扯什么响应■■政策需要，这个 B 东西是默认打开的而且无法关闭，是的无法关闭，虽然提供了后端环境变量但只要登录了管理员帐号就一定会在前端看到发送人的 IP，如果不改前端源码就只能用 CSS display: none 来达成眼不见心不烦。
　　如果这玩意不是开源 + 自建后端我一定把它拉黑……
